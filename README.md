# Matter of Space

A premium luxury interior design website recreated with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, GSAP, Swiper.js, and Lenis.

## Setup

1. Install dependencies

```bash
npm install
```

2. Run local development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

## Features

- Next.js App Router with TypeScript
- Tailwind CSS design system
- Smooth scroll with Lenis
- Scroll-triggered animation with GSAP
- Responsive luxury UI sections
- Reusable components and CMS-ready static data
- Dark mode support

## Structure

- `src/app` — pages and layout
- `src/components` — reusable UI components
- `src/sections` — homepage and page sections
- `src/data` — static CMS-ready content
- `src/lib` — animations and utilities
- `src/styles` — global styling
- `public/images` — placeholder assets
